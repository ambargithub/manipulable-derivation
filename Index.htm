Derivation of motion of a string

1. String description

Let us assume that we have an unstreched string tied at both ends and we wish to derive the motion of the string when it is stretched. Which means that we wish to know how any particle of the string behaves whenever it is streched

Drag the slider to view different string positions

2. Discretization

Let’s start by assuming that this string is composed of numerous small mass points which are linked to each other by springs. The distance b/w the two mass points is dx.

3. Arbitrary mass point

Let’s have a look at an arbitrary mass point \(m_{i}\), which is the \(i^{th}\) element on the string. Let’s say that \(m_{i}\) is located at \(x_{i}\) and \(y_{i}\) on the coordinate system.

4. Arbitrary mass point

In a streched position, the spring which connects \(m_{i}\) to \(m_{i-1}\) will also be streched. mi would therefore experience a force of \( F_{i-1}( x_{i})\) due to mi-1. Let's calculate this force:

\( F_{i-1}( x_{i})\) = \(\ -k \)\(( strech\ in\ the\ spring)\)
               = \(\ -k \) ( \(hypotenuse\) - \(dx\))

5. Modeling the force

\(F_{i-1}( x_{i})\) \( =\ k_{1}\) \(\sqrt{}\) { \(( dY_{i-1})^{2}\) \(\ +\ \) \(( dx)^{2}\)} \(- dx\)

                \(F_{i+1}( x_{i})\) \({{{{=\ k_{1} \ ( Y_{(}}_{x}}_{i+1)} \ -\ Y_{(}}_{x}}_{i)})\)
